# C# 数据库 DEMO
编程环境：
- Windows 10
- Visual Studio 2019 community （C#）
- SQLserver 2017 Developer edition

----------------------------

学习顺序：

1. calculator 简单计算器，熟悉 控件 和 编程。

2. Login 学习连接数据库

3. DBdemo 学习通过设置数据源，在datagridview中显示数据库中的数据。

4. CRUD 学习增删改查操作。代码较多，但操作并不复杂。

5. School MDI窗口类型。
