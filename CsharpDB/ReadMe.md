# C# 数据库 DEMO
编程环境：
- Windows 10
- Visual Studio 2017 community （C#）
- SQLserver 2017 Developer edition

----------------------------
DBdemo 学习通过设置数据源，在datagridview中显示数据库中的数据。

LoginDemo 学习登录部分代码。此例程完成了增删改查，但操作较复杂，建议熟练VS后再学习。

CRUD 学习增删改查操作。代码较多，但操作并不复杂。
